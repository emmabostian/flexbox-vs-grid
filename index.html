CSS Flexbox vs. CSS Grid

Emma Wedekind

Flexbox

Flexbox

  • Provides an efficient way to lay out, align, and distribute space among items in a container.
  • Container has the ability to alter items' size in order to fill available space.
  • Layout is direction-agnostic - flexible.

Grid

Grid

  • Used to create a 2-dimensional layout.
  • Works in tandem with CSS Flexbox.
  • Source order of the grid-items doesn't matter - easy to re-arrange for different screen sizes.

Terminology

Grid Container

The element on which `display: grid` is applied.

Terminology

Grid Item

The children of the grid container; items are grid-items but sub-items are not.

Terminology

Grid Line

The dividing lines that make up the structure of the grid. They can be either vertical ("column grid lines") or horizontal ("row grid lines") and reside on either side of a row or column.

Terminology

Grid Track

The space between two adjacent grid lines. You can think of them like the columns or rows of the grid.

Terminology

Grid Cell

A single unit on the grid.

Terminology

Grid Area

The total space surrounded by four grid lines. A grid area may be comprised of any number of grid cells.
